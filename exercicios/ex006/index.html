Principais Formatações

Negrito / Destaque

(NÃO SEMÂNTICA) Temos um termo em negrito usando a tag <b>

(SEMÂNTICA) Temos um termo em destaque usando a tag <strong>

Itálico / Ênfase

(NÃO SEMÂNTICA) Temos um termo em itálico usando a tag <i>

(SEMÂNTICA) Temos um termo em ênfase usando a tag <em>

Texto Marcado

(SEMÂNTICA) Temos um termo marcado usando a tag <mark>

Texto Grande e Pequeno

Temos um (OBSOLETO) texto grande e um (SEMÂNTICA) texto pequeno usando as tags <big> (aumente na formatação) e <small>

Texto Deletado

(SEMÂNTICA) Temos um texto marcado como texto exluido para indicar que deve ser lido mas não considerado usando a tag <del>

Sublinhado / Inserido

(NÃO SEMÂNTICA) Temos um termo sublinhado usando a tag <u>

(SEMÂNTICA) Temos um termo inserido para dar ênfase e indicar que foi adicionado depois usando a tag <ins>

Texto Sobrescrito

(SEMÂNTICA) Temos um termo tipo x20+3 usando a tag <sup>

Texto Subscrito

(SEMÂNTICA) Temos um termo tipo H20 usando a tag <sub>
